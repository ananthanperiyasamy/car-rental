## To start 
To run the frontEnd angular component:
1. Go to the folder "Frank-warehouse-frontend"
2. Run "npm install" to package
3. Run command ->  ng serve --port 8081



To Kill Node.js:
-----------------
ps aux | grep node
Kill all process id's starts with 814 "kill -9 814**"
